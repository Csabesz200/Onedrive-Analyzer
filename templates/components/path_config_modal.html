Common OneDrive paths and how to find yours

Common OneDrive paths:

  • Personal: C:\Users\YourName\OneDrive
  • Business: C:\Users\YourName\OneDrive - CompanyName

To find your OneDrive folder path:

  1. Open File Explorer
  2. Navigate to your OneDrive folder
  3. Right-click on the address bar and select "Copy address as text"
  4. Paste the copied path in the field above
